# Anymotion

Anymotion provides one unified API for animations with support for UIKit, CoreAnimation, POP and your library of choice

### Installation

```
pod 'Anymotion'
```

##### Import in swift

```
import Anymotion
```

##### Import in Objective-C

```
#import <Anymotion/Anymotion.h>
```
